Custom Select can be customized in css by customizing .custom-select-wrapper class for select box and .custom-select-list for the drop-down option.

Tested in IE 9/10//11 , Microsoft Edge, Chrome, Firefox and Opera(latest).

Steps:
1) Copy custom-select.css and customselect.js file from css and js folder respectively.
2) Link to jquery, css and js file:
	<link rel="stylesheet" href="css/custom-select.css">
	<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
	<script src="js/custom-select.js"></script>
3) Put this at the end of the html body:
	<script>
		var options = {
			customSelectClass : "select-browser", //enter class name of the select 
								you want to customize
			borderColor: "red" //optional bottom-border color
		};
		$('.example-wrapper').CustomSelect(options);	
	</script>
